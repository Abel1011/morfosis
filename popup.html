Mórfosis

Configure the options for rewriting, summarizing, suggesting, and translating text. Then select the text you want to process and click on the Mórfosis icon in the browser toolbar. Alternatively, you can press Ctrl + M to open the chat.

Rewrite

Configures the options for rewriting the text.

Summarize

Configures the options for summarizing the text.

Suggestions

Configures the options for the suggestions.

Translate

Configures the options for translating the text.
